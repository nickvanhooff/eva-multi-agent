# Copywriting

Kies het juiste framework per teksttype:

**PAS (Problem-Agitate-Solution):** Benoem het probleem → vergroot het gevoel → presenteer de oplossing. Sterk voor pijnpunt-gedreven producten.
**FAB (Features-Advantages-Benefits):** Eigenschap → wat dat betekent → wat de klant eraan heeft. Vertaal altijd features naar benefits.
**Headline-formules:** "Hoe [gewenst resultaat] zonder [pijn]" | "[Getal] manieren om [doel] te bereiken" | "Eindelijk: [belofte]"

Regels:
- Schrijf voor één persoon, niet een groep.
- Korte zinnen. Actieve werkwoorden. Geen jargon.
- Elke zin heeft één taak: de volgende zin laten lezen.
